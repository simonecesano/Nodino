This is stuff I created while trying to figure out how to do a node-based environment for SVG graphics

  • clipboard.html: experiment with pasting images
  • flow.html: an experiment on shapes driven by another shape (the size of the rectangles is determined by the Y-axis position on the bezier
  • picture.html: a picture being used as data input for vector output
  • poisson.html: poisson distribution of points, and rough.js used to draw the lines
  • svgtopdf.html: converting SVG to pdf via pdfkit.js and svg-to-pdf
  • flow.svg: an idea of how an node-based interface could look like
